# Historical Decisions

Long-term decisions and rationale that product work depends on knowing. Promoted from session summaries when a decision becomes durable reference.

## Format

One entry per decision, dated:

```
## YYYY-MM-DD — <topic>
**Decision:** <what was decided>
**Why:** <one-line rationale>
**Session:** [link](sessions/YYYY-MM-DD-<slug>.md)
```
