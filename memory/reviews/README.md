# Reviews — Periodic Pattern Analysis

Written by the `review-memory` skill. Reviews analyze recent sessions to identify repeated workflows, manual patterns, and automation opportunities.

## Structure

Files are named `YYYY-MM-DD.md`. Each review covers a window of prior sessions (default: 30 days).

## When to use

- Deciding what to automate next
- Reviewing whether automation investments are paying off
- Checking for skill gaps or recurring toil

## When NOT to use

- Single-session recap → `memory/sessions/`
- Half-formed ideas → `memory/staging/`
