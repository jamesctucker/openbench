# Work Repos Registry

Persistent agent context about repos cloned into `work/`. Update when you clone, remove, or significantly change a work repo.

## Format

One entry per repo:

```
### <repo-name>
- **Stack:** <languages, frameworks>
- **Purpose:** <one line>
- **AGENTS.md:** [link](../work/<repo-name>/AGENTS.md)
- **Notes:** <conventions, gotchas, refs>
```
